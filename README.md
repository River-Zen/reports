# Harness 实战手册

静态教程站，记录文件驱动的 Agent 工作流与真实项目实践。

## 页面

- `index.html`：教程首页
- `harness.html`：Harness 从 0 到正式接管
- `three-agents.html`：三 Agent 协作闭环
- `tao-universe.html`：道德经宇宙搭建实战

旧版 `Harness搭建教程.html` 与 `Harness体系-20260708.html` 保留，避免已有链接失效。

## 本地预览

```powershell
python -m http.server 8000
```

浏览器打开 `http://127.0.0.1:8000/`。

## 发布

仓库使用纯静态 HTML/CSS/JS，可直接从 `main` 分支根目录发布到 GitHub Pages，无构建依赖。
